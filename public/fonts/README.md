# Title font: Melodrama

The **Melodrama** font (Indian Type Foundry) is loaded from `melodrama/` in this folder and used for all titles. The app uses these files:

- `Melodrama-Variable.ttf` – variable font (300–700)
- `Melodrama-Light.otf`, `Melodrama-Regular.otf`, `Melodrama-Medium.otf`, `Melodrama-Semibold.otf`, `Melodrama-Bold.otf` – static weights

If these files are missing, the app falls back to **Bodoni Moda** (Google Fonts) for titles.
